Welcome to the Admin Dashboard

This dashboard allows administrators to manage and monitor attendance-related activities effectively. Use the buttons above to navigate between key sections:

  • Attendance Edit Requests: Review and approve/reject requests submitted by teachers for modifying attendance records.
  • Manage Students: View, add, update, or remove student profiles. Each student is associated with a batch and can have attendance entries.
  • Manage Teachers: Manage teacher records including assigning them to batches and giving them access to attendance functionalities.

Please use the navigation buttons above to get started.
